double xdg::BoundingBox::dilation ( ) const
inline

Returns a distance by which the box should be dilated to account for mixed precision effects.

This method calculates a small distance that can be used to dilate the bounding box to account for potential numerical precision issues when working with mixed precision calculations. The distance is based on the maximum chord length of the box and the precision of float values.
